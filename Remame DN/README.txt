С помощью скрипта есть возможность переименовать пользователей/ботов указав из какого файла брать имена 
(в файлах указываються имена на кириллице/латинеце/китайские иероглифы и прочее)

в rename_bots_DN.py указать адрес сервера его токен, префикс id пользователей (общаяя часть id ) 
и из какого файла брать имена (так же возможно потребуется указать диапазаон для файла 
в скрипте, это место отмечено коментарием)
