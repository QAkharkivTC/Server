Даные скрипты для тестов в ЛК пользователя

Перед запуском необходимо найти строку наподобие этой:
url = 'https://10.130.2.63/api/v3.1/templates/conferences/?access_token=e139525ea73f9776ed75485c8c94b7e8140ecb4c'

где необходимо указать ip или домен сервера, id пользователя и токен со страницы личного кадинета (через dewtools при выполнении какого либо запроса)
